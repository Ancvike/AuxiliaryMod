本模组一些功能代码借鉴了Sharlottes/Informatis, MinRi2/AutoSaver模组

2.无限制缩放

3.蓝图大小无限制

4.AI敌人进攻路线

5.战争迷雾的开启与关闭

6.单位数量无限制

7.当前单位和所控制单位无敌

8.查看敌人波次

9.核心资源最大化

10.己方建筑修复

11.己方单位修复

12.清除废墟

13.传送带替换

14.自由的区块发射

15.改变队伍
